\# FoodeX



FoodeX is a full-stack online food ordering and restaurant management platform developed using React.js, ASP.NET Core Web API, Entity Framework Core, and SQL Server.



\## Features



\### Customer

\- User Registration and Login

\- Browse Restaurants

\- Search Restaurants and Products

\- Shopping Cart

\- Order Placement

\- Favorite Restaurants

\- Restaurant Reviews and Ratings

\- Profile Management



\### Restaurant Administrator

\- Restaurant Dashboard

\- Product Management

\- Stock Management

\- Order Management

\- Customer Reviews

\- Restaurant Settings



\### Super Administrator

\- Restaurant Application Management

\- Restaurant Approval and Rejection

\- User Management

\- Restaurant Management



\### Security

\- JWT Authentication

\- BCrypt Password Hashing

\- First Login Password Change

\- Role-Based Authorization



\### Technologies



\#### Frontend

\- React.js

\- React Router

\- CSS3

\- Vite



\#### Backend

\- ASP.NET Core Web API

\- Entity Framework Core

\- SQL Server



\#### Additional Libraries

\- MailKit

\- BCrypt.Net
